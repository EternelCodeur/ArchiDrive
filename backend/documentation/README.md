# Documentation backend (Laravel)

## Objectif

Cette documentation est générée à partir du code source du dossier `backend/`.

## Index

- Les documents détaillés par fichier sont générés dans les sous-dossiers de `backend/documentation/`.
- Pour régénérer automatiquement la documentation, utiliser le script `scripts/generate-docs.mjs` à la racine du dépôt.

## Convention

- 1 fichier Markdown par fichier source.
- Le chemin du document de doc suit le chemin du fichier source.
