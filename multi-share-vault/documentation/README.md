# Documentation frontend (multi-share-vault)

## Fichiers principaux

- `frontend.md` : documentation générale de l’application.

## Documentation par fichier

Les documents détaillés par fichier sont générés dans les sous-dossiers de `multi-share-vault/documentation/`.

Pour régénérer automatiquement la documentation : exécuter `node scripts/generate-docs.mjs` depuis la racine du dépôt.
